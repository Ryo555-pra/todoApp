## 目的

1. **todo アプリの自分好みのカスタム**  
   既存のアプリでも多く todo 管理を行うものはありますが、外部アプリ（LINE、Slack 等）と連携したい、達成率の管理を可視化したいなど、自分の好みにカスタムしてみたいと思い作ることを決めました。

2. **実装スキルの向上・様々な機能の実践の場**  
   正直、こちらの意味合いが強いかなと思っています。  
   その上で、自分の欲しい機能（1 の内容）を実装していきたいと考えています。
